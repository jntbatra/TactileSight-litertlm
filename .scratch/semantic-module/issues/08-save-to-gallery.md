# Save to gallery (double-press)

Status: ready-for-agent

## What to build

Double-press saves the current Scene Frame to the phone gallery and plays the **"captured" earcon** — the only deliberate path to persistence (ephemeral-by-default otherwise).

## Acceptance criteria

- [ ] Double-press → a photo of the current frame appears in the phone gallery.
- [ ] A distinct "captured" earcon plays.
- [ ] Nothing else is persisted.

## Blocked by

- One-button gesture disambiguation
